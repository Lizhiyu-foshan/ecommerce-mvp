## 🔧 修复方案详情

### BUG-004: 库存扣减并发控制

**问题**: 多个用户同时购买时可能导致超卖

**解决方案**:
```python
# 方案1: 悲观锁
SELECT * FROM products WHERE id = ? AND stock >= ? FOR UPDATE

# 方案2: 乐观锁
UPDATE products SET stock = stock - ?, version = version + 1 
WHERE id = ? AND version = ?

# 方案3: 原子操作
UPDATE products SET stock = stock - ? WHERE id = ? AND stock >= ?
```

**预期效果**: 消除超卖问题，保证库存一致性
