| 项目 | 内容 |
|------|------|

| **测试目的** | 验证用户能够成功创建一条非默认的收货地址 |

### 数据库状态检查SQL

```sql
-- 查询用户地址数量
SELECT COUNT(*) FROM addresses WHERE user_id = 'user_id';

-- 查询用户默认地址
SELECT * FROM addresses WHERE user_id = 'user_id' AND is_default = 1;

-- 查询用户所有地址（按默认优先、时间倒序）
SELECT * FROM addresses 
WHERE user_id = 'user_id' 
ORDER BY is_default DESC, created_at DESC;
```
